# UX Flows

## Onboarding
1. Land on marketing page with login.
2. Choose login provider.
3. On first login, see empty state with subscribe CTA.

## Subscribe to a Feed
1. User enters a URL.
2. System detects if URL is a feed or a website.
3. If website, show discovered feeds and let user pick.
4. Success state shows feed added and first items fetched.

## Reading Flow
1. User selects a feed or tag in left navigation.
2. Feed items appear as cards in a vertical stream.
3. Card includes title, source, time, snippet, and actions.
4. Clicking card expands inline or opens a focused reader panel.
5. User scrolls to next card without table navigation.

## Save and Tag
1. User clicks Save on a card.
2. Prompt shows existing tags with autocomplete.
3. User selects or adds tags.
4. Saved item appears in Saved view and is cached in R2.

## Annotation Flow
1. In saved item view, user selects text.
2. Inline tool offers Highlight or Comment.
3. Highlight wraps selection; comment creates a note anchored to selection.
4. Comments appear in the margin or inline popovers.

## Tag Management
1. Tag autocomplete shows recent and matching tags.
2. Tag management allows rename and delete.
3. Tag removal from a feed or saved item is one click.

## Search and Filters (Future)
1. Search within saved items only.
2. Filter by tag, source, or date.
