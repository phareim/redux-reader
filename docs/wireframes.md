# Wireframes (Textual)

## Desktop Layout
- Left sidebar with logo, search, tags, feeds.
- Main column with card stream.
- Right panel optional for saved item annotations.

```
+-------------------------------------------------------------+
| Logo   Search                            Profile            |
+---------+----------------------------------+----------------+
| Tags    |  Card 1 (title, source, snippet) | Notes/Comments  |
| Feeds   |  Card 2                          | Highlight list  |
| Saved   |  Card 3                          | Tag editor      |
|         |  ...                             |                |
+---------+----------------------------------+----------------+
```

## Mobile Layout
- Bottom nav for Feeds, Saved, Tags, Profile.
- Card stream full width.
- Annotation drawer slides from bottom.

```
+-------------------------+
| Logo        Profile     |
+-------------------------+
| Card 1                  |
| Card 2                  |
| Card 3                  |
+-------------------------+
| Feeds  Saved  Tags  Me  |
+-------------------------+
```

## Card Anatomy
- Title
- Source + time
- Snippet or cover image
- Actions: Save, Tag, Mark Read
